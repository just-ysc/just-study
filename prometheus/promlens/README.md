# PromLens

- PromLens는 PromLabs에서 개발한 웹 기반 PromQL 빌더이다.
    - PromQL 쿼리의 작성, 작성된 쿼리에 대한 설명 및 분석, 시각화를 지원한다.
- 최초에 상업용으로 제작되었으나, 이후에 오픈 소스로 기증되었다.

## How to Install

- 바이너리를 다운로드 받아서 서버에 직접 설치할 수 있고, 도커 이미지를 다운로드 받아 띄울 수도 있다.
- 자세한 설치 방법은 TBU
